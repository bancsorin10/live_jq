# live_jq

**live_jq** is a terminal user interface for interactively previewing [jq](https://jqlang.github.io/jq/) query results against JSON input. Type a jq query and see the result update live, with syntax-colored output.

## Vibecoded

This entire app was vibecoded in ~20 minutes using [opencode](https://opencode.ai) — an AI coding assistant. Zero hand-written code; every line was generated by AI agents.

## Usage

```
live_jq [FILE] [-n <lines>]
```

Read JSON from a file, or pipe it via stdin:

```
curl https://api.example.com/data | live_jq
```

- `-n <lines>` — minimum height of the query input area (default: 1, auto-expands as you type)

### Key bindings

| Key | Action |
|-----|--------|
| `Esc` | Quit |
| `Tab` | Toggle focus between output and query |
| `↑` / `↓` | Scroll output |
| `PgUp` / `PgDn` | Scroll output by page |

## Install

### GitHub Actions

Download the binary from the **Build Release** workflow artifacts:

1. Go to Actions → Build Release → Run workflow
2. Download the artifact after it completes
3. Make executable: `chmod +x live_jq-x86_64-linux/live_jq`

### Build from source

```
cargo build --release
./target/release/live_jq
```
